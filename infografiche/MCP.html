Model Context Protocol

Lo Standard Universale 2026

Sostituisce le integrazioni custom con un'unica interfaccia client-server

Host / Client

Esempi: Claude Desktop, IDE, Agent Frameworks.

  • Gestisce l'autenticazione
  • Esegue l'orchestrazione
  • Fornisce l'interfaccia utente
JSON-RPC 2.0 OVER SSE/STDIO
Scambio di messaggi bidirezionale

MCP Server

Esempi: GitHub Server, DB Server, Local Files.

  • Espone dati e funzioni
  • Esegue la logica lato tool
  • Isola l'accesso ai dati sensibili

Cosa offre un Server MCP?

Tools

Funzioni eseguibili che l'LLM può invocare (Write access).

list_tools()
call_tool("create_jira_issue")

Resources

Dati statici o dinamici leggibili (Read access).

list_resources()
read_resource("mcp://db/logs")

Prompts

Template predefiniti per istruire l'LLM su task specifici.

get_prompt("analyze_log")
-> returns system instructions

Flusso di Lavoro 2026

01
Configurazione

Aggiungi l'URL del server o il comando nel file config dell'host.

02
Discovery

L'LLM interroga il server per sapere quali tools e risorse sono disponibili.

03
Execution

L'utente chiede un task; l'AI seleziona ed esegue il tool tramite MCP.
